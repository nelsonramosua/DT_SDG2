# Design Thinking — Repositório de Desenvolvimento do Trabalho
---
# Overview
Este repositório documenta o desenvolvimento do projeto no âmbito do módulo de **Design Thinking** da unidade curricular CT1, com foco no **Objetivo de Desenvolvimento Sustentável 2 (SDG2 – Zero Hunger)**.  
Inclui recursos e registos do processo criativo, desde a fase de empatia até à ideação e prototipagem.
Não será de forma nenhuma utilizado para efeitos de entrega; tão-só para facilitar o desenvolvimento.
---

## Objetivo Geral

Compreender as necessidades, sentimentos e motivações de diferentes intervenientes relacionados com o problema da **fome e desperdício alimentar**, de modo a fundamentar a definição de oportunidades e soluções inovadoras.

O mapa sintetiza os contributos recolhidos nas entrevistas a três personas representativas do contexto:

- **Maria (42 anos)** – mãe solteira, enfrenta insegurança alimentar.  
- **João (33 anos)** – dono de restaurante, lida com desperdício alimentar.  
- **Ana (25 anos)** – voluntária de associação local de apoio alimentar.
